O.U.F.T.I.

Our User Friendly Thermals Interface.

This project was designed for helping pilots to analyse thermals and potential XC pathways for a given take-off. Data exploited by O.U.F.T.I. were gathered from XContest and Syride public data. In a later phase the tool will be released as an open source project to allow users of other take-offs to implement it.
